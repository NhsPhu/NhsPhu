### 🎬 Movie Streaming System — *Dự án trọng tâm*

> Hệ thống phát video **Adaptive Streaming (HLS)** đa nền tảng theo kiến trúc **Microservices**.

- 🏗️ **Backend:** Java Spring Boot · API Gateway · Spring Data JPA
- 📱 **Mobile:** React Native (Web + Mobile đồng bộ real-time)
- 🤖 **AI Service:** Python FastAPI — gợi ý phim cá nhân hóa bằng **Collaborative Filtering (Scikit-learn)**
- 🎞️ **Media:** FFmpeg xử lý encode/transcoding video tự động sang HLS
- 🔐 **Auth:** JWT + Spring Security · Swagger API docs
- 🐳 **DevOps:** Docker · Docker Compose

---

### 📊 AI Sales Analysis Dashboard

> Hệ thống **Business Intelligence** tự động hóa phân tích dữ liệu bán hàng.

- 📥 **ETL Pipeline:** Tự động trích xuất và chuẩn hóa dữ liệu từ CSV/JSON
- 📈 **Visualization:** Dashboard KPI tương tác với Plotly
- 🤖 **AI:** Tích hợp Groq API — truy vấn dữ liệu bằng ngôn ngữ tự nhiên
- 🔮 **Forecast:** Dự báo xu hướng doanh thu với thuật toán Facebook Prophet
